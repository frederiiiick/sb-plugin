Installation

Follow the steps below to setup Showing Bee plugin:

  1. Import the following script into your HTML file after the body tag. You can use either the normal or minified version of the script 
                    <script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin/dist/sb-plugin-alpha.js"></script>
<!-- Minified Version ============================================= -->
<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin/dist/sb-plugin-alpha.min.js"></script>
  2. Add the sb-plugin element with the prefered id and mls number like this on the HTML body 
                    <sb-plugin id="example-id" data-mls-number="T3404758"></sb-plugin>
  3. Initialize the script with the id used on the sb-plugin element 
                    <script type="text/javascript">initSBPlugin('example-id');</script>
  4. You are good to go! You should be able to see a button like this if done successfully

Data Attributes

You can add data attributes to change how the plugin will work and display on the web page

  • data-mls-number Required - declade the MLS Number of the listing
  • data-size - specify the size of the button, default is normal (choices: small, normal, large)
  • data-agent-request - indicate if the link would go to for agent request page or listing page, default is false (choices: true, false)
  • data-title - change button title, default is Showing Bee
  • data-extra-css - add extra inline css styling to the button, make sure to seperate the styling with a semi-colon

Simply add the attributes in the <sb-plugin> element.
Make sure to add the id and data-mls-number on the element to make the plugin work, since these both are required attributes.

For Example:

<sb-plugin 
  id="example-id"
  data-mls-number="T3404758"
  data-size="small"
  data-agent-request="true"
  data-title="View on Showing Bee"
  data-extra-css="background: #2a2a2a;"
></sb-plugin>
