Installation

Follow the steps below to setup Showing Bee plugin:

  1. Import the following script into your HTML file after the body tag. You can use either the normal or minified version of the script 
                    <script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin@alpha-v2.1/dist/sb-plugin-alpha.js"></script>
<!-- Minified Version ============================================= -->
<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin@alpha-v2.1/dist/sb-plugin-alpha.min.js"></script>
  2. Add the sb-plugin element with the prefered id and mls number on the HTML body like this 
                    <sb-plugin id="example-id" data-mls-source="1" data-mls-number="T3435873" data-agent-info='{"email":"info@getmoreoffers.com", "first_name":"Keith", "last_name":"Gordon", "phone_number":"+12025556789", "company":"GetMoreOffers"}'></sb-plugin>
  3. Initialize the script with the id used on the sb-plugin element 
                    <script type="text/javascript">initSBPlugin('example-id');</script>
  4. You are now good to go! You should be able to see a button if done correctly

Data Attributes

You can add data attributes to change how the plugin will work and display on the web page

  • data-mls-source Required - declare the MLS Source of the listing
  • data-mls-number Required - declare the MLS Number of the listing
  • data-agent-info Required - declare the Agent Data for the listing. Must be in JSON format.
  • data-size - specify the size of the button, default is normal (choices: small, normal, large)
  • data-title - change button title, default is Showing Bee
  • data-window - change how the plugin will showing on the browser, default is popup (choices: popup, tab)
  • data-env - change endpoint environment, default is prod (choices: prod, dev)
  • data-extra-css - add extra inline css styling to the button, make sure to seperate the styling with a semi-colon

Simply add the attributes in the <sb-plugin> element.
Make sure to add the id, data-mls-source, data-mls-number and data-agent-info on the element to make the plugin work, since these are the required attributes.

For Example:

<sb-plugin 
  id="example-id"
  data-mls-source="3"
  data-mls-number="A11402797"
  data-agent-info='{"email":"info@getmoreoffers.com", "first_name":"Keith", "last_name":"Gordon", "phone_number":"+12025556789", "company":"GetMoreOffers"}'
  data-size="small"
  data-title="View on Showing Bee"
  data-extra-css="background: #2a2a2a;"
></sb-plugin>
