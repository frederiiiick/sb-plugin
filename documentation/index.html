Installation

Follow the steps below to setup Showing Bee plugin:

  1. Import the following script into your HTML file after the body tag. You can use either the normal or minified version of the script 
                    <script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin/dist/sb-plugin-alpha.js"></script>
<!-- Minified Version ============================================= -->
<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/frederiiiick/sb-plugin/dist/sb-plugin-alpha.min.js"></script>
  2. Add the sb-plugin element with the prefered id and mls number like this on the HTML body 
                    <sb-plugin id="example-id" data-mls-number="T3404758"></sb-plugin>
  3. Initialize the script with the id used on the sb-plugin element 
                    <script type="text/javascript">initSBPlugin('example-id');</script>
  4. You are good to go for adding your content now!

Color Schemes

You can change your Website's Color Scheme in an instant. We have build 10 Color css file for change template primary color.

  • color-blue.css - Blue Color
  • color-brown.css - Brown Color
  • color-cyan.css - Cyan Color
  • color-green.css - Green Color
  • color-orange.css - Orange Color
  • color-indigo.css - Indigo Color
  • color-purple.css - Purple Color
  • color-red.css - Red Color
  • color-teal.css - Teal Color
  • color-yellow.css - Yellow Color

Simply add Color CSS file link it in the Document <head>.
Make sure you add the css/color-blue.css stylesheet in your head after the css/stylesheet.css file linking from above mentioned 10 color css file.

For Example:

<head>
<!-- Stylesheet
============================== -->
<!-- Bootstrap -->
<link rel="stylesheet" type="text/css" href="assets/vendor/bootstrap/css/bootstrap.min.css" />
<!-- Font Awesome Icon -->
<link rel="stylesheet" type="text/css" href="assets/vendor/font-awesome/css/all.min.css" />
<!-- Magnific Popup -->
<link rel="stylesheet" type="text/css" href="assets/vendor/magnific-popup/magnific-popup.min.css" />
<!-- Highlight Syntax -->
<link rel="stylesheet" type="text/css" href="assets/vendor/highlight.js/styles/github.css" />
<!-- Custom Stylesheet -->
<link rel="stylesheet" type="text/css" href="assets/css/stylesheet.css" />

<!-- Here goes your Colors Css -->
<link rel="stylesheet" type="text/css" href="assets/css/color-orange.css" />
</head>
